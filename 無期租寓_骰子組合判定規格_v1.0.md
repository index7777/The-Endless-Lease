# 《無期租寓》骰子組合判定規格 v1.0

## 固定輸入

六顆六面骰依序對應：體魄、反應、意志、感知、交涉、異常親和。提交不需要點選單顆骰子；裁定完成後即可提交，或選擇重新擲骰。

## 最高優先隱藏結果

- `6,6,6,6,6,6`：觸發隱藏職業及該職業專屬天賦，天然機率 `1 / 46,656`。
- `1,1,1,1,1,1`：觸發另一個隱藏職業及該職業專屬天賦，天然機率 `1 / 46,656`。
- 兩組正式名稱、敘述與效果等待企劃文件補齊；此前不得自行命名或杜撰。
- 隱藏結果優先於所有一般身份與天賦判定。

## 一般身份

一般身份由最高的兩項能力組合決定；同點時依六項固定順序處理，確保相同骰面永遠得到相同結果。完整對照表實作於 `app/game/destiny-rules.ts`。

## 一般天賦

依序判定：異常親和＋感知、交涉＋意志、反應＋交涉、體魄＋意志；皆未達門檻時取得「租緩」。先命中的規則生效，沒有額外隨機抽選。
