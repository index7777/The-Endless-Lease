# B2 清剿完成至 Demo 結束：完整開發規格

## 0. 開發目標

將目前「打倒 B2 後立刻顯示普通完成彈窗」的流程，重構成具有商業遊戲品質的 Demo 收尾段落。

此段落必須完成以下目的：

1. 讓玩家感受到擊敗 B2 對公寓造成了實際變化。
2. 給予玩家具體獎勵及清楚的任務收尾。
3. 讓管理員紅怡重新參與劇情。
4. 預告不存在於樓層表中的 10F。
5. 讓玩家有短暫整理、探索及存檔時間。
6. 以世界內事件結束 Demo，而不是突然跳出「Demo 完成」視窗。
7. 留下正式版的核心懸念。

不得使用以下內容：

- 「以新住戶再次入住」
- B2 死亡後直接返回標題
- 普通「關卡完成」或「恭喜通關」文案
- 單純顯示「10F 尚未開放」
- 玩家未實際拾取道具，系統直接宣告獲得
- 沒有紅怡參與的 Demo 結局
- 只用一張靜態文字面板完成全部收尾

本段落的核心情緒不是勝利，而是：

> 玩家解決了地下室的問題，卻意外讓更危險的樓層注意到自己。

# 1. B2 戰鬥結束演出

## 1.1 B2 死亡後的基本流程

B2 生命歸零後依序執行：

1. 停止玩家攻擊輸入，但保留攝影機控制。
2. 使用 0.1 至 0.2 秒的極短時間縮放，強調最後一次衝擊。
3. 播放 B2 死亡動畫。
4. B2 不得立即消失。
5. 戰鬥音樂逐步移除打擊樂與高頻。
6. 播放地下管線、金屬結構與電力系統重新啟動的聲音。
7. 場景燈光短暫全部熄滅。
8. 緊急燈、遠端走廊燈與電梯燈依序恢復。
9. 電梯方向傳來一次不屬於 B1 或 B2 的到站聲。
10. B2 遺體或核心產生「特殊權限卡」。
11. 恢復玩家操作。
12. 顯示情境任務：「拾取異常物件」。

不得在 B2 死亡後立刻跳出完成面板。

16. 音樂規格

需要製作或整理三個音樂／氛圍段落。

16.1 B2 戰鬥收尾

長度約 20 至 40 秒。

不使用英雄式勝利旋律。
保留低頻殘響。
加入不穩定的電力與金屬聲。
10F 按鍵亮起時加入一個新的高音動機。
16.2 返回管理室

以環境音為主，不使用完整勝利音樂。

內容：

遠處電梯聲。
六樓低頻管線聲。
管理室電話或紙張聲。
將 10F 高音動機藏在背景中。

長度約 45 至 75 秒。

情緒曲線：

安靜
→ 電梯機械低頻
→ 三次敲門
→ 紅怡電話警告
→ 10F 主題動機
