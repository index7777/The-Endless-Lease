# 《無期租寓》電影化演出與互動最高規範 v1.0

最後更新：2026-07-16
效力：最高

本文件的完整正式內容位於根目錄 `AGENTS.md` 的「附錄 III：《無期租寓》電影化演出與互動規範 v1.0」。該附錄是所有 Codex 任務的強制規則，必須完整閱讀，不得只讀本入口摘要。

適用於所有劇情事件、管理室事件、UI 演出、文件閱讀、NPC 對話、特殊發現、異常事件、房間探索、玩家控制、鏡頭、音效與轉場。凡演出與其他文件衝突，以 `AGENTS.md` 附錄 III 為準。

不可刪減的最高原則：

- 所有資訊與 UI 必須具有世界內實體來源。
- 玩家控制只能處於 Gameplay、Administrative Event、Cinematic 三種狀態。
- 文件、公告與管理程序必須使用完整鏡頭、取放、閱讀及恢復控制流程。
- 重要正式資訊回到獨立管理室行政中心處理。
- 紅怡演出冷靜、優雅、緩慢並保留停頓。
- 鏡頭慢、穩、克制；音效由世界而非 UI 主導。
- 異常來自行政程序逐步出錯，不以 Jump Scare 取代。
- 七張 approved 圖是唯一 UI 真值，只能使用既有 Design System 元件重組。
- 無法使用既有元件完成時，停止實作並提出衝突，不得自行設計新 UI。
- 任何修改完成後仍需製作人實機驗收。
