## 其他類別

- `characters/`：尚無已批准角色比例基準；不得用聊天圖自行補入。
- `animation/`：尚無通過 Animation Validator 的正式動畫基準。
- `environments/`：待從既有定案場景建立正式索引。
- `lighting/`：待建立場景光照基準。
- `audio/`：待建立音訊聽感與響度基準。
- `rejected/`：保存被取代或明確拒絕、但需要追溯的素材。
