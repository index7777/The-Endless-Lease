# 《無期租寓》六骰獨立判定規格 v2.0

## 固定輸入

六顆六面骰依序為 `d1…d6`，對應體魄、反應、意志、感知、交涉、異常親和。六顆骰面全部參與正式運算；沒有六個 1 或六個 6 的隱藏判定。

## 三套獨立公式

- 身分：`SN = d1 + d3 + d5`，`N = (SN - 1) mod 6 + 1`
- 天賦：`ST = d2 + d4 + d6`，`T = (ST - 1) mod 6 + 1`
- 缺陷：`SD = d1 + d2 + d3 + d4 + 2d5 + 2d6`，`D = (SD - 1) mod 6 + 1`

缺陷採配對加權總和，取代會使 D6 機率達 89.80% 的六骰乘積。完整枚舉 46,656 種骰面後：N、T、D 各結果皆為精確 `1/6`；三者組成的 216 種聯合結果各出現 216 次，因此亦為統計獨立。

## 固定對照

- 身分：1 清潔工、2 醫師、3 逃犯、4 房仲、5 靈媒、6 前住戶
- 天賦：1 租緩、2 窺層、3 偽居、4 契語、5 蝕耐、6 存契
- 缺陷：1 貪息、2 寡眠、3 幻聽、4 直白、5 輕厄、6 無家

範例 `[1,2,3,4,5,2]`：N=3、T=2、D=6，因此為「逃犯＋窺層＋無家」。

## 提交流程

裁定完成後可直接提交，或選擇重新擲出六顆骰子；不要求點選或鎖定單顆骰子。
